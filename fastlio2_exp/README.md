# Unitree G1 Humanoid - FastLio2 implementation

## Overview
Here are the scripts I ran in the robot in order to record a rosbag and then executing the fastlio algorithm in my PC

![Fastlio pointcloud](../assets/fastlio.png)

## Scripts
There are 3 scripts, one for being able to read the pointcloud, on for the realsense (using the Unitree's SDK) and another for using an addicional webcam.
In order to work, it is important to set the correct IPs for the lidar. I don't quite remember how I did it, but if it is of your interest, submit an issue and I can search for it again.
